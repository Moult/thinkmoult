This is a 5-panel window, with three on the left and two on the right. The window frame is wooden and painted white. There is a single pane of clear glazing.

The windows panes on the left are awning windows, which open outwards. It has latch hardware to lock the window in place.

Materials

The colours have been calibrated with Macbethcal but the specularity and roughness values are guessed.

The glazing is approximated using glaze as a single clear pane.

Usage

This Radiance model follows the Radiance Filesystem Hierarchy Standard. As such, please run make lib to build the Radiance mesh file. Once done, you can use this in your scene by:

!xform lib/window-5-panel-right-1530w-2230h/obj/model.rad
